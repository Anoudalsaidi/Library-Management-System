The Library Management System allows community members to register and access library services. Once registered, a member can borrow books that have available copies in stock. Each book contains information such as its title, author, genre, and the total number of physical copies available. When a book is borrowed, the system creates a loan record, stores the borrowing date, and automatically sets a due date 14 days later. Upon returning the book, the system updates the available copies and checks whether the return was made after the due date. If the book is returned late, a fine is calculated based on the number of overdue days. If all copies of a book are currently borrowed, members cannot borrow it immediately but may place a reservation request. When a copy becomes available, the system notifies the next member in the reservation queue. Librarians can monitor all active loans, view overdue loans sorted by the number of days overdue, and generate reports that rank books according to their total borrow count, helping identify the most frequently borrowed books in the library.
